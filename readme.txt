== Description ==

Displays your favorite plugins (in a widget) from your WordPress.org profile along with star ratings and author info.


== Installation ==

* Activate
* go to appearance -> widgets
* drag Favorite Plugins widget to sidebar of choice
* enter wordpress.org username
* profit
